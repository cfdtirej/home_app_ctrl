> # get_room_state.py
室内環境データを取得して記録するプログラム.  
- NatureRemoAPIから時刻time・名前name・MacAddress・湿度hu・温度te・照度ilを取得
- 湿度・温度から不快指数di・体感温度tmを計算
- それらのデータをCSV & InfluxDBに記録
> # calc_di_tm.py
不快指数DIと体感温度TMを計算する  
id_lebelは温度設定の変更
|不快指数|感覚|di_lebel|
| ---- | ---- | ---- |
|-55|寒い|2|
|55-60|肌寒い|1|
|60-65|何も感じない|0|
|65-70|快適|0|
|70-75|熱くない|0|
|75-80|やや暑い|-1|
|80-85|暑い|-2|
|85-|クソ暑い|-3|
